# Discovery Analysis Style

## Output
- Lead with the finding. Context after.
- Bullets over prose paragraphs.
- No narrative fluff between sections.

## Accuracy Rules
- If important information is missing: say so.
- If confidence is low: state it explicitly with a reason.
- Distinguish clearly between what the user said and what is inferred.
- Label inferences explicitly.

## Report Format
- Summary first.
- Supporting detail second.
- Caveats, open questions, and limitations last.

## Simple Formatting
- No em dashes or smart quotes.
- Natural language characters are fine when the content requires them.
